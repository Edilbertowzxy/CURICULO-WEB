📄 Currículo Online

Este projeto consiste no desenvolvimento de um currículo online interativo, criado utilizando HTML, CSS e JavaScript, com o objetivo de praticar conceitos fundamentais de desenvolvimento web e apresentar minhas informações profissionais de forma moderna e acessível.

🚀 Sobre o Projeto

Como estudante de Sistemas de Informação, estou constantemente buscando evoluir minhas habilidades em programação e desenvolvimento web. Durante meus estudos, desenvolvi este projeto para aplicar na prática os conhecimentos em estruturação de páginas com HTML, estilização com CSS e interatividade com JavaScript.

O currículo online permite apresentar minhas informações profissionais de forma organizada, acessível e visualmente agradável diretamente no navegador.

🛠️ Tecnologias Utilizadas

HTML5 – Estrutura da página

CSS3 – Estilização e layout

JavaScript – Interatividade e funcionalidades dinâmicas

📚 Objetivos do Projeto

Praticar conceitos de desenvolvimento web

Aprender sobre estrutura semântica em HTML

Trabalhar com estilização e layout com CSS

Implementar funcionalidades interativas com JavaScript

Criar um currículo digital acessível online

📌 Funcionalidades

Apresentação de informações pessoais

Seção de habilidades

Formação acadêmica

Layout responsivo e organizado

Interações básicas com JavaScript

🎯 Aprendizados

Durante o desenvolvimento deste projeto, foi possível reforçar conhecimentos como:

Estrutura de páginas com HTML

Manipulação básica do DOM com JavaScript

Organização de conteúdo para aplicações web
